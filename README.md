# Flask Blog

Pretty simple blog written with Flask. I used few tecnologies and framework for this simple project.
*  Bootstrap 5
*  Peewee (ORM Database for SQLite)
*  CKEditor

## Usage
```bash
python -m pip install -r requirements.py
python main.py
```
